As remembering VK_NUMPAD0 is less easy that NP0
The following file near the Jar allows to add shortcut ot it.

For example if you want to use Zero PadZero to type zero from OMI.
You can do it like this 

`KeyShortcut.txt`
```
VK_NUMPAD0: PadZero
VK_0: Zero
```
